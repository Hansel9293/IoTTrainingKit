Introductory Project 3

ESP32 PWM (Analog Output)

ABOUT THE PROJECT

In this tutorial, we’ll guide you through generating PWM (Pulse Width Modulation) signals with the ESP32 using the Arduino IDE. As an example, we’ll create a simple circuit to control the brightness of an LED using the ESP32’s built-in LED PWM controller.

PARTS NEEDED:

• Main Board Module
• LED
• 220 Ohm Resistor
• Breadboard
• Jumper Wires

THE BUILD

INSTRUCTION(s):

The ESP32 features a LED PWM controller with 16 independent channels that can each be configured to generate unique PWM signals. Follow these steps to dim an LED using PWM with the Arduino IDE:

1. Choose a PWM Channel: Select one of the 16 available channels, numbered 0 through 15, to generate the PWM signal.

2. Set the PWM Frequency: Define the frequency of the PWM signal. For controlling an LED, a frequency of 5000 Hz is suitable.

3. Configure Duty Cycle Resolution: Specify the resolution of the PWM signal, which can range from 1 to 16 bits. We’ll use an 8-bit resolution, allowing you to adjust LED brightness with values between 0 and 255.

4. Assign GPIO for Output: Indicate the GPIO pin that will output the PWM signal. To do this, use the following function:

      ledcAttachPin(GPIO_PIN, PWM_CHANNEL);
      - GPIO_PIN is the pin number where the LED is connected.
      - PWM_CHANNEL is the channel generating the signal.

5. Control LED Brightness: Use the function below to set the duty cycle and adjust LED brightness:

     ledcWrite(PWM_CHANNEL, dutyCycle);
      - PWM_CHANNEL is the channel generating the PWM signal.
      - dutyCycle (from 0 to 255) determines the LED’s brightness, with higher values making the LED brighter.

By following these steps, you’ll be able to control the LED’s brightness smoothly using PWM on the ESP32.

We'll program the ESP32 using the Arduino IDE, so ensure that you have the ESP32 add-on installed before continuing. (If you’ve already completed this setup, you can skip this step.)

After uploading the code, press the ESP32 reset button, then open the Serial Monitor at a baud rate of 115200. As you rotate the potentiometer, you should see the values change in real-time on the Serial Monitor.

ESP32 Project

CODE

 

// LED pin assignment
const int ledPin = 4; // GPIO 4

// PWM properties
const int freq = 5000;      // Frequency of the PWM signal
const int ledChannel = 0;   // Channel to generate the PWM signal
const int resolution = 8;   // 8-bit resolution (0-255)

void setup() {
  // Configure LED PWM functionality
  ledcSetup(ledChannel, freq, resolution);

  // Attach the PWM channel to the LED pin
  ledcAttachPin(ledPin, ledChannel);
}

void loop() {
  // Increase LED brightness
  for (int dutyCycle = 0; dutyCycle <= 255; dutyCycle++) {
    // Set the PWM duty cycle
    ledcWrite(ledChannel, dutyCycle);
    delay(15);  // Adjust delay to control fade speed
  }

  // Decrease LED brightness
  for (int dutyCycle = 255; dutyCycle >= 0; dutyCycle--) {
    // Set the PWM duty cycle
    ledcWrite(ledChannel, dutyCycle);
    delay(15);  // Adjust delay to control fade speed
  }
}
