Project 1

Leave a Reminder

HOW IT WORKS?

In this leve a reminder project, you will learn how to use an ultrasonic sensor, a buzzer, and a web server to create a system built for sending message and sound alerts. Upon entering desired message or a reminder, the system will start working. It will start sending the message and alarm soon as the sensor detect any movement.

PARTS NEEDED:

• Main Board Module
• Motion Module
• Connector Cable

SENSOR USED IN THE MODULE

HC-SR04 Ultrasonic Sensor
This module is used in the project in such a way that it emits sound waves that travel through the air. Threshold is placed in the code for distancing what will be considered a movement.

INSTRUCTION(s):

Step 1: Assembling the System
Positioning the Module:
1. Take the Motion Module (enclosed in the 3D-printed box) and align it over the main board.
2. Gently press the module onto the main board's connector pins until it fits securely in place.
3. Ensure that the module is correctly seated to avoid loose connections.

Step 2: Powering the System
Connecting Power:
1. Get the connecting cable and attach it into the laptop that you will use to upload the code.
2. Plug in the other end of it to the main board. The system will automatically power on.
Visual Indicator Check:
Observe the LED indicators on the main board. The system should show a steady power LED to confirm it's operational.

Step 3: Connecting to Web Server:
1. Once the serial monitor says "Code Uploaded", open the serial monitor of the Arduino IDE then press the EN button to show the IP address.
2. Open your WiFi and look for ESP32_Reminder. Choose that option and enter password 12345678. 3. Enter the IP address in a search engine 192.168.4.1.

Step 4: Monitoring and Alerts
Understanding the Alerts:
1. The system is equipped with a visual (web server) alert.
2. When the web server loads, it prompts the user to enter a reminder message. Monitoring begins once a message is entered. If movement is detected, the system displays the message on the screen and activates a buzzer to alert the person nearby that a reminder has been left for them. They can turn off the alert by pressing the button below the message.

Testing the Alerts:
To test the system, place you palm in front of the sensor. The alerts should activate.

CODE

 

#include <WiFi.h>
#include <WebServer.h>

// Define the Access Point credentials
const char* ssid = "ESP32_Reminder";
const char* password = "12345678";

// Define Static IP Address
IPAddress local_IP(192, 168, 4, 1);
IPAddress gateway(192, 168, 4, 1);
IPAddress subnet(255, 255, 255, 0);

// Ultrasonic Sensor Pins
const int trigPin = 5;   // Trig pin for the ultrasonic sensor
const int echoPin = 18;  // Echo pin for the ultrasonic sensor

// Buzzer Pin
const int buzzerPin = 4; // Pin for the buzzer

#define SOUND_SPEED 0.034
long duration;
float distanceCm;
const float activationThreshold = 30.0; // Activation distance threshold in cm (1 foot)
String reminderMessage = ""; // Reminder message set by the user
bool reminderActive = false;
bool messageEntered = false; // Tracks if a reminder message has been entered

WebServer server(80); // Create a web server on port 80

// Function to read the distance from the ultrasonic sensor
float readDistance() {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  duration = pulseIn(echoPin, HIGH, 30000); // 30ms timeout to avoid hanging
  float distance = duration * SOUND_SPEED / 2;  // Convert time to distance in cm

  // Check for valid distance
  if (distance <= 0 || distance > 400) {
    return -1; // Invalid distance
  }
  return distance;
}

// Function to check if the reminder should be triggered
void checkReminder() {
  // Do not start if no message has been set
  if (!messageEntered) {
    return;
  }

  // If already active, do not check further
  if (reminderActive) {
    return;
  }

  distanceCm = readDistance();

  // If the user is detected within the specified activation range, activate the reminder
  if (distanceCm > 0 && distanceCm < activationThreshold) {
    reminderActive = true;
    digitalWrite(buzzerPin, HIGH); // Turn on the buzzer
    Serial.println("Reminder Activated: " + reminderMessage);
  }
}

// Serve the web page based on the current state
void handleRoot() {
  String html = "<html><head><title>Air Quality Reminder</title>";
  html += "<meta name='viewport' content='width=device-width, initial-scale=1'>";
  html += "<style>";
  html += "body { font-family: Arial, sans-serif; text-align: center; background: radial-gradient(circle, white, #00ab41); color: black; height: 100vh; display: flex; justify-content: center; align-items: center; margin: 0; }";
  html += ".container { width: 90%; max-width: 400px; padding: 20px; text-align: center; }";
  html += "h1 { font-size: 2.5rem; margin-bottom: 20px; }";
  html += ".status-bar { font-size: 1.2rem; margin-top: 20px; }";
  html += ".large-message { font-size: 5rem; color: red; }";
  html += "button { margin-top: 20px; padding: 10px 20px; font-size: 1.5rem; cursor: pointer; }";
  html += "</style>";
  html += "<script>";
  html += "var refreshInterval;";
  html += "function startRefreshing() {";
  html += "refreshInterval = setInterval(updatePage, 1000);";
  html += "}";
  html += "function stopRefreshing() {";
  html += "clearInterval(refreshInterval);";
  html += "}";
  html += "function updatePage() {";
  html += "fetch('/status').then(function(response) {";
  html += "return response.json();";
  html += "}).then(function(data) {";
  html += "if (data.reminderActive) {";
  html += "document.body.innerHTML = '<div class=\"container\"><div class=\"large-message\">' + data.reminderMessage + '</div><button onclick=\"turnOffAlarm()\">Turn Off Alarm</button></div>';";
  html += "} else if (data.messageEntered) {";
  html += "document.body.innerHTML = '<div class=\"container\"><h1>Reminder Status</h1><div class=\"status-bar\">Reminder is Inactive</div>';";
  html += "} else {";
  html += "document.body.innerHTML = '<div class=\"container\"><h1>Enter Reminder Message</h1><input type=\"text\" id=\"messageInput\" placeholder=\"Enter reminder message\" onfocus=\"stopRefreshing()\" onblur=\"startRefreshing()\"><button onclick=\"updateMessage()\">Set Reminder Message</button></div>';";
  html += "}";
  html += "});";
  html += "}";
  html += "function updateMessage() {";
  html += "var message = document.getElementById('messageInput').value;";
  html += "fetch('/setMessage?message=' + encodeURIComponent(message)).then(function() {";
  html += "startRefreshing();";
  html += "});";
  html += "}";
  html += "function turnOffAlarm() {";
  html += "fetch('/turnOffAlarm');";
  html += "}";
  html += "window.onload = startRefreshing;";
  html += "</script>";
  html += "</head><body>";

  html += "<div class='container'>";
  html += "<h1>Loading...</h1>";
  html += "</div>";

  html += "</body></html>";

  server.send(200, "text/html", html);
}

// Handle setting the reminder message
void handleSetMessage() {
  if (server.hasArg("message")) {
    reminderMessage = server.arg("message");
    messageEntered = true; // Indicate that a message has been entered
    server.send(200, "text/plain", "Message set to: " + reminderMessage);
  } else {
    server.send(400, "text/plain", "Invalid request");
  }
}

// Handle turning off the alarm
void handleTurnOffAlarm() {
  reminderActive = false;
  digitalWrite(buzzerPin, LOW); // Turn off the buzzer
  server.send(200, "text/plain", "Alarm turned off");
}

// Serve the reminder status as JSON for AJAX requests
void handleStatus() {
  String json = "{ \"reminderActive\": " + String(reminderActive ? "true" : "false") + 
                ", \"messageEntered\": " + String(messageEntered ? "true" : "false") + 
                ", \"reminderMessage\": \"" + reminderMessage + "\" }";
  server.send(200, "application/json", json);
}

void setup() {
  Serial.begin(115200);

  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  pinMode(buzzerPin, OUTPUT);
  digitalWrite(buzzerPin, LOW); // Ensure buzzer is off initially

  // Start WiFi as an access point
  WiFi.softAPConfig(local_IP, gateway, subnet);
  WiFi.softAP(ssid, password);

  Serial.println("Access Point Started");
  Serial.println(WiFi.softAPIP());

  // Start the web server
  server.on("/", handleRoot);
  server.on("/setMessage", handleSetMessage);
  server.on("/turnOffAlarm", handleTurnOffAlarm);
  server.on("/status", handleStatus);
  server.begin();
}

void loop() {
  server.handleClient();  // Handle web requests
  checkReminder();        // Check if the reminder should be triggered
}

CODE EXPLANATION

This code sets up a Multipurpose Reminder System using an ultrasonic sensor and a buzzer connected to an ESP32. The system detects a user’s proximity and triggers a reminder with a custom message when they are within a specified distance. Users can connect to an Access Point (AP) and interact with the system through a web interface to set up reminders for various purposes, such as checking air quality, closing windows, or any other configurable notification. Let’s break it down.

Key Components and Their Roles
1. Libraries Included:
   •WiFi.h Executes Wi-Fi related activities, thus enabling the ESP32 to configure the formation of an Access Point for use by a user.
2. WiFi Setup
   • By inserting the ESP32, an Access Point is configured with the Service Set Identifier (SSID) ESP32_Reminder and the password 12345678.
   • The ESP32 configures a static IP address for easy connection to the control interface by users.

3. Ultrasonic Sensor and Buzzer Setup:
   • The Trig and Echo of the ultrasonic sensor are connected to GPIO pins 5 and 18 of the Arduino system respectively. The sensor is distance based which works by estimating the amount of time the sound waves take before they are reflected back.
   • The buzzer is connected to GPIO pin 4 and sounds when a user comes within a specified distance (activation threshold) and the reminder is active.
   • TheactivationThresholdis set to 30 cm (1 foot), which triggers the reminder if a user approaches within this range.

4. Server Functions:
   • Key server functions include:
     •handleRoot()Showcases the home page through which users can send out a reminder note or check the reminder on. This function changes the content according to whether there is an active reminder or if there is a need for a particular message setting.
     •handleSetMessage()Recovers a message that is received from the user through the Web interface and saves it for the reminder.
     •handleTurnOffAlarm()Enables the user to neutralise the alarm whenever it is active.

5. Data Collection withreadDistance():
   • ThereadDistance()function is able to determine the distance at which the sensor is from the object that it has detected.
   • It makes the buzzer go off and displays the reminder message in the Web page if an object is in the range of the activationThreshold

6. Web Interface Design:
   • The interface uses HTML and CSS to create a clean, functional display.
   • Thecontainerdiv structures the page, showing either the reminder message or an input for setting a message if none exists.
   • When the reminder is active, a large, bold message appears, and the user can turn off the alarm through a button.

7. JavaScript and AJAX Updates:
   • JavaScript polls the/statusendpoint every second, fetching reminder status and updating the display without needing a page refresh.
   • Users can set the reminder message or turn off the alarm directly through the interface, making it easy to interact with the system.

8. The Setupsetup()and Looploop()Functions:
   •setup()Initializes the ultrasonic sensor pins and buzzer, starts the Access Point, and configures the web server.
   • loop()Continuously handles client requests, checks for user presence throughcheckReminder(), and updates the web interface in real-time.
