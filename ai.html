Introductory Project 2

ESP32 Analog Inputs

ABOUT THE PROJECT

In this project, you will be learning about how an ESP32 is able to reag analog inputs using different variable resistors.

PARTS NEEDED:

• Main Board Module
• Ptentiometer
• Breadboard
• Jumper Wires

THE BUILD

INSTRUCTION(s):

The VCC and GND of the potentiometer should ceonnected to the corresponding ESP32 pins while the signal pin is connected to D4.

ESP32 Project

CODE

 

// Potentiometer is connected to GPIO 4 (Analog
ADC2_CH0) const int potPin = 4;

// variable for storing the potentiometer valueint potValue = 0;

void setup() {
Serial.begin(115200);
delay(1000);}

void loop() {
// Reading potentiometer value
potValue = analogRead(potPin);
Serial.println(potValue);
delay(500);}
