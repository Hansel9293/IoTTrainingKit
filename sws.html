Introductory Project 4

ESP32 Switch Web Server

ABOUT THE PROJECT

In this project, you’ll build a standalone web server using the ESP32 and the Arduino IDE, allowing you to control two LEDs from any device with a browser on your local network. The web server is designed to be mobile responsive, so it works well on both desktop and mobile devices. We’ll guide you through creating the web server and explain the code step-by-step.

Before diving into the project, here’s a summary of what our web server will accomplish to help make the steps easier to follow:
   - LED Control: The web server will control two LEDs connected to the ESP32’s GPIO 26 and GPIO 27 pins.
   - Accessing the Web Server: You’ll be able to access the ESP32 web server by typing the ESP32’s IP address into a browser on any device within the same local network.
   - Interactive Control: The web interface will feature buttons that allow you to toggle each LED on or off in real time by simply clicking.
With this setup, you’ll have a responsive web server that controls outputs, providing a straightforward way to manage connected devices from anywhere within your network.

PARTS NEEDED:

• Main Board Module
• 2 x LED
• Breadboard
• Jumper Wires
• 2 x 220 Ohm Resistor

THE BUILD

INSTRUCTION(s):

To begin, build the circuit by connecting two LEDs to the ESP32, following the schematic diagram:
   • Connect one LED to GPIO 26: Attach the positive (anode) leg of the LED to GPIO 26 and the negative (cathode) leg to ground via a resistor (typically 220Ω or 330Ω to limit current).
   • Connect the other LED to GPIO 27: Similarly, connect the positive leg of this LED to GPIO 27 and the negative leg to ground with a resistor.

Note: This guide uses the ESP32 DEVKIT DOIT board with 36 pins. Before wiring, ensure you verify the pinout for your specific board model, as pin configurations may vary. This helps prevent miswiring and ensures that each LED is correctly connected to the intended GPIO pins.

ESP32 Project

CODE

 

#include <WiFi.h>

// Replace with your network credentials
const char* ssid = "YOUR_SSID";
const char* password = "YOUR_PASSWORD";

// Pin assignments for LEDs
const int ledPin1 = 26;  // GPIO 26
const int ledPin2 = 27;  // GPIO 27

// Variable to store LED states
bool ledState1 = false;
bool ledState2 = false;

// Create the WiFiServer object on port 80
WiFiServer server(80);

void setup() {
  // Initialize serial communication
  Serial.begin(115200);

  // Set up the LED pins as outputs
  pinMode(ledPin1, OUTPUT);
  pinMode(ledPin2, OUTPUT);

  // Connect to Wi-Fi network
  WiFi.begin(ssid, password);
  Serial.print("Connecting to WiFi");
  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }
  Serial.println();
  Serial.println("Connected to WiFi");
  Serial.print("IP Address: ");
  Serial.println(WiFi.localIP());

  // Start the server
  server.begin();
}

void loop() {
  // Check if a client has connected
  WiFiClient client = server.available();
  if (!client) {
    return;
  }

  // Wait until the client sends some data
  while (!client.available()) {
    delay(1);
  }

  // Read the request line
  String request = client.readStringUntil('\r');
  client.flush();

  // Control LEDs based on the request
  if (request.indexOf("/LED1/ON") != -1) {
    ledState1 = true;
  }
  if (request.indexOf("/LED1/OFF") != -1) {
    ledState1 = false;
  }
  if (request.indexOf("/LED2/ON") != -1) {
    ledState2 = true;
  }
  if (request.indexOf("/LED2/OFF") != -1) {
    ledState2 = false;
  }

  // Set LED states
  digitalWrite(ledPin1, ledState1);
  digitalWrite(ledPin2, ledState2);

  // HTML response for the web page
  String html = "<!DOCTYPE html><html><head><title>ESP32 Web Server</title>";
  html += "<meta name=\"viewport\" content=\"width=device-width, initial-scale=1.0\">";
  html += "<style>body { font-family: Arial; text-align: center; } button { padding: 20px; font-size: 20px; margin: 10px; }</style></head>";
  html += "<body><h1>ESP32 Web Server</h1>";
  html += "<p>LED 1 - State: " + String(ledState1 ? "ON" : "OFF") + "</p>";
  html += "<p><a href=\"/LED1/ON\"><button>Turn LED 1 ON</button></a>";
  html += "<a href=\"/LED1/OFF\"><button>Turn LED 1 OFF</button></a></p>";
  html += "<p>LED 2 - State: " + String(ledState2 ? "ON" : "OFF") + "</p>";
  html += "<p><a href=\"/LED2/ON\"><button>Turn LED 2 ON</button></a>";
  html += "<a href=\"/LED2/OFF\"><button>Turn LED 2 OFF</button></a></p>";
  html += "</body></html>";

  // Send the response to the client
  client.println("HTTP/1.1 200 OK");
  client.println("Content-type:text/html");
  client.println("Connection: close");
  client.println();
  client.println(html);

  // Close the connection
  client.stop();
}
