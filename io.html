Introductory Project 1

ESP32 Input Output

ABOUT THE PROJECT

For the said purposes, in this introductory guide, you will be informed of how to use ESP32 with Arduino IDE in reading digital inputs such as a button switch and control digital outputs like an LED. For the above purposes, we are going to make a simple project using a pushbutton and an LED. In this example, we are going to read the state of the pushbutton connected and control the LED lighting or not based on the pressed button as depicted in the figure.

PARTS NEEDED:

• Main Board Module
• LED
• 10k Ohm Resistor
• Breadboard
• Jumper Wires
• Pushbutton
• 220 Ohm Resistor

THE BUILD

INSTRUCTION(s):

Before you begin, you will establish a circuit with an LED and a pushbutton. Connect this circuit up to your ESP32, having your LED connected at GPIO 5 and your push button at GPIO 4. Upload the code above. After uploading, you should look at your circuit. Your LED should now come on each time you click on your pushbutton.

ESP32 Project

CODE

 

// Pin assignments
const int buttonPin = 4; // the number of the pushbutton pin
const int ledPin = 5;    // the number of the LED pin

// Variable for storing the pushbutton state
int buttonState = 0;

void setup() {
  // Start the serial communication for debugging
  Serial.begin(115200);
  
  // Initialize the pushbutton pin as an input
  pinMode(buttonPin, INPUT);
  
  // Initialize the LED pin as an output
  pinMode(ledPin, OUTPUT);
}

void loop() {
  // Read the state of the pushbutton value
  buttonState = digitalRead(buttonPin);

  // Check if the button is pressed
  if (buttonState == HIGH) {
    // Turn on the LED
    digitalWrite(ledPin, HIGH);
  } else {
    // Turn off the LED
    digitalWrite(ledPin, LOW);
  }
}
